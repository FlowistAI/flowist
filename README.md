# Vite React Flow Template

This template creates a very basic [React Flow](https://reactflow.dev) app with [Vite](https://vite.dev).

## Get it!

```sh
npx degit xyflow/vite-react-flow-template app-name
```

## Installation

```sh
npm run install
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```
